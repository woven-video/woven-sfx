# Transition pairings

Default sound IDs per transition type. The MCP server applies this logic in `sfx_resolve` — use these as reference only.

| Transition | Primary sound ID |
|------------|------------------|
| `pull-in` | `swish-whoosh-large` |
| `pull-out` | `fast-whoosh` |
| `flash` | `camera-flash` |
| `glitch` | `glitch-logo` |
| `tv-glitch` | `glitch-logo` |
| `whoosh` | `fast-whoosh` |

**No default SFX:** `fade`, `none`

**Opening whoosh:** place `whoosh` at **0.1s** on the timeline.
